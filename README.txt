# Projet Django - Gestion des Tâches

## Étapes pour démarrer :

1. Crée un environnement virtuel :
   python -m venv env

2. Active l'environnement :
   - Windows : env\Scripts\activate
   - Linux/macOS : source env/bin/activate

3. Installe Django :
   pip install django

4. Va dans le dossier :
   cd gestion_taches_web_v2

5. Lance les migrations :
   python manage.py migrate

6. Démarre le serveur :
   python manage.py runserver

7. Va sur :
   http://127.0.0.1:8000/

- Pour créer un compte : /register/
- Pour te connecter : /
- Pour te déconnecter : /logout/
